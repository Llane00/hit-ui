# Hit-ui

PC UI Components built on React.

You can use very simple code to run every component.

Have Fun! :)

[Hit-ui website](https://jetdream.xyz/hit-ui/)

## install:

`yarn`

## run:

`yarn dev`

## build:

`yarn build`

## test command:

```
yarn test
yarn test -u //update snapshots
yarn test --watch //run test server
yarn ci
```
